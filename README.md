# make-swap-memory-vps
Membuat Swap memory di VPS linux agar dengan menu pilihan dan bisa permanent tetap aktif sata reboot.

Jalankan dengan perintah
```bash
wget https://raw.githubusercontent.com/ica4me/make-swap-vps/main/make-swap.sh && chmod +x make-swap.sh && bash make-swap.sh
```

Verifikasi
```bash
free -h
```
